## Product Entity

The `Product` entity represents a product in the billing system.

### Properties

<ParamField path="id" type="String" required>
  Unique identifier for the product. Usually the barcode, but kept separate for flexibility.
</ParamField>

<ParamField path="name" type="String" required>
  Product name
</ParamField>

<ParamField path="barcode" type="String" required>
  Product barcode for scanning
</ParamField>

<ParamField path="price" type="double" required>
  Product price
</ParamField>

<ParamField path="stock" type="int" default="0">
  Current stock quantity
</ParamField>

### Constructor

```dart
const Product({
  required String id,
  required String name,
  required String barcode,
  required double price,
  int stock = 0,
})
```

```dart

  GetProductsUseCase(this.repository);

}
```

## ProductBloc

### Constructor

```dart

  required AddProductUseCase addProductUseCase,
  required UpdateProductUseCase updateProductUseCase,
  required DeleteProductUseCase deleteProductUseCase,
```

### State

<ResponseField name="status" type="ProductStatus" required>
  Current status: `initial`, `loading`, `loaded`, `error`, or `success`
</ResponseField>

<ResponseField name="products" type="List<Product>" default="[]">
  List of loaded products
</ResponseField>

<ResponseField name="message" type="String?">
  Success or error message
</ResponseField>

```dart
enum ProductStatus { initial, loading, loaded, error, success }

class ProductState extends Equatable {
  final ProductStatus status;
  final List<Product> products;
  final String? message;

  const ProductState({
    this.status = ProductStatus.initial,
    this.products = const [],
    this.message,
  });

  ProductState copyWith({
    ProductStatus? status,
    List<Product>? products,
    String? message,
  });
}
```
